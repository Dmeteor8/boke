# CommitmentSection 组件

服务承诺展示区块组件，用于展示企业的服务承诺和保障。

## 功能特性

- ✅ 渲染四项服务承诺
- ✅ 突出显示核心承诺文本
- ✅ 使用图标增强视觉效果
- ✅ 响应式网格布局（移动端 1 列，平板 2 列，桌面 4 列）
- ✅ 滚动进入视口时的渐显动画
- ✅ 卡片悬停效果

## Props

### CommitmentSectionProps

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `commitments` | `Commitment[]` | 是 | 服务承诺列表 |
| `highlight` | `string` | 是 | 核心承诺文本（突出显示） |

### Commitment

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | `string` | 是 | 承诺唯一标识 |
| `text` | `string` | 是 | 承诺文本内容 |
| `icon` | `string` | 否 | 图标（当前版本未使用，图标由组件自动分配） |

## 使用示例

### 基本用法

```tsx
import { CommitmentSection } from './components/CommitmentSection';

const commitments = [
  {
    id: '1',
    text: '项目按约定周期推进',
  },
  {
    id: '2',
    text: '功能确认后再开发',
  },
  {
    id: '3',
    text: '提供上线指导与基础维护',
  },
  {
    id: '4',
    text: '可签订合作协议',
  },
];

const highlight = '每个项目都以长期稳定运行为目标';

<CommitmentSection
  commitments={commitments}
  highlight={highlight}
/>
```

## 设计说明

### 布局结构

1. **区块标题**：居中显示"服务承诺"标题和副标题
2. **核心承诺**：使用渐变背景卡片突出显示核心承诺文本
3. **承诺网格**：响应式网格展示四项具体承诺

### 响应式断点

- **移动端** (< 768px)：1 列布局
- **平板端** (768px - 1024px)：2 列布局
- **桌面端** (> 1024px)：4 列布局

### 图标映射

组件自动为每个承诺分配图标：

- **索引 0**：时钟图标 - 适合"项目按约定周期推进"
- **索引 1**：检查列表图标 - 适合"功能确认后再开发"
- **索引 2**：工具图标 - 适合"提供上线指导与基础维护"
- **索引 3**：文档图标 - 适合"可签订合作协议"

### 动画效果

- **标题动画**：从下方淡入（opacity + translateY）
- **核心承诺动画**：缩放淡入（opacity + scale）
- **承诺卡片动画**：依次从下方淡入，每个卡片延迟 0.1 秒
- **悬停效果**：卡片阴影增强

### 颜色方案

- **背景**：蓝色到靛蓝的渐变（from-blue-50 to-indigo-50）
- **核心承诺卡片**：蓝色到靛蓝的渐变（from-blue-600 to-indigo-600）
- **图标背景**：浅蓝色（bg-blue-100）
- **图标颜色**：蓝色（text-blue-600）

## 注意事项

1. **承诺数量**：组件设计为展示 4 个承诺，但可以支持任意数量
2. **图标循环**：如果承诺数量超过 4 个，图标会循环使用
3. **文本长度**：建议每个承诺文本保持简洁（10-20 字）
4. **核心承诺**：highlight 文本应该简洁有力，建议不超过 30 字
