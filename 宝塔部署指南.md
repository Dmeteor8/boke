# 宝塔面板部署 Next.js 项目指南

## 一、准备工作

### 1. 服务器要求
- Linux 系统（Ubuntu/CentOS）
- 内存至少 1GB
- 已安装宝塔面板

### 2. 安装宝塔（如未安装）

**Ubuntu:**
```bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
```

**CentOS:**
```bash
wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh
```

## 二、安装必要软件

### 1. 登录宝塔面板
访问 `http://服务器IP:8888`

### 2. 安装软件
在「软件商店」安装：
- **PM2 管理器**（会自动安装 Node.js）
- **Nginx**

## 三、上传项目

### 方式一：Git 克隆（推荐）
```bash
cd /www/wwwroot
git clone 你的项目地址
```

### 方式二：手动上传
1. 在宝塔「文件」中进入 `/www/wwwroot`
2. 创建项目文件夹
3. 上传项目文件

## 四、安装依赖和构建

```bash
cd /www/wwwroot/你的项目名
npm install
npm run build
```

## 五、配置 PM2

### 1. 创建配置文件
```bash
nano ecosystem.config.js
```

### 2. 添加配置
```javascript
module.exports = {
  apps: [{
    name: 'my-website',
    script: 'npm',
    args: 'start',
    cwd: '/www/wwwroot/你的项目名',
    env: {
      NODE_ENV: 'production',
      PORT: 3000
    }
  }]
};
```

### 3. 启动项目
```bash
pm2 start ecosystem.config.js
pm2 save
pm2 startup
```

## 六、配置 Nginx

### 1. 添加网站
- 宝塔面板 → 网站 → 添加站点
- 域名：你的域名
- 根目录：项目目录

### 2. 配置反向代理
- 点击网站「设置」
- 反向代理 → 添加
- 目标 URL：`http://127.0.0.1:3000`

### 3. 优化配置（可选）
点击「配置文件」，添加：

```nginx
location / {
    proxy_pass http://127.0.0.1:3000;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_cache_bypass $http_upgrade;
}
```

## 七、配置 SSL（HTTPS）

1. 网站设置 → SSL
2. 选择「Let's Encrypt」
3. 申请证书
4. 勾选「强制 HTTPS」

## 八、开放端口

在「安全」中开放：
- 80（HTTP）
- 443（HTTPS）

## 九、验证部署

访问你的域名，检查网站是否正常运行。

## 常用命令

```bash
# 查看状态
pm2 status

# 查看日志
pm2 logs

# 重启
pm2 restart my-website

# 停止
pm2 stop my-website
```

## 更新代码

```bash
cd /www/wwwroot/你的项目名
git pull
npm install
npm run build
pm2 restart my-website
```

## 故障排查

### 网站无法访问
```bash
pm2 logs  # 查看错误日志
pm2 restart my-website  # 重启应用
```

### 端口被占用
```bash
netstat -tunlp | grep 3000  # 检查端口
```

### 内存不足
升级服务器配置或增加交换空间

## 完成！

现在你的 Next.js 网站已经成功部署到宝塔面板了。
